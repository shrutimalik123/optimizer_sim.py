# ✈️ The Gradient Glider: Neural Network Optimizer Sim

An interactive Machine Learning simulation designed to teach **Gradient Descent**, the mathematical engine behind how Neural Networks learn. You play as a pilot navigating through a thick fog, using only the "slope" of the ground beneath you to find the lowest point in the valley (the Global Minimum).

## 🎓 Learning Objectives

This project focuses on teaching:
* **Gradient Descent:** The optimization algorithm used to minimize a loss function by iteratively moving in the direction of steepest descent[span_0](start_span)[span_0](end_span).
* **Learning Rate (Alpha):** A hyperparameter that determines the size of the steps taken toward the minimum[span_1](start_span)[span_1](end_span).
* **The Loss Function:** Understanding how mathematical functions (like $y = x^2$) represent "Error" in a model[span_2](start_span)[span_2](end_span).
* **Overshooting:** Visualizing what happens when a learning rate is too high, causing the model to diverge[span_3](start_span)[span_3](end_span).
* **Convergence:** The process of reaching a stable point where the error is minimized[span_4](start_span)[span_4](end_span)[span_5](start_span)[span_5](end_span).

---

## ✨ Features

* **Aviation Scenario:** Simplifies the abstract concept of "finding the bottom of a curve" into a relatable landing mission.
* **Dynamic Step Calculation:** Uses the derivative of the function to calculate the gradient in real-time.
* **Hyperparameter Experimentation:** Allows users to manually set the Learning Rate to see the immediate impact on convergence speed and stability.
* **Visual Feedback:** Provides step-by-step telemetry showing current position, gradient, and total error (MSE).
